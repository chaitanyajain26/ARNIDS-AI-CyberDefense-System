# ARNIDS - AI Powered Cyber Defense System

ARNIDS is an AI-powered adversarial robust network intrusion detection system designed for real-time traffic analysis, attack detection, and cybersecurity risk assessment.

## Features

- Real-time packet capture
- AI-based intrusion detection
- Open port vulnerability analysis
- LLaMA-powered attack explanation
- Voice-enabled AI assistant
- Adversarial robustness testing

## Tech Stack

- Python
- Machine Learning
- Random Forest
- Logistic Regression
- Wireshark / tshark
- Nmap
- LLaMA (LM Studio)
